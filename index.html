سياسة الخصوصية واستخدام البيانات

البيانات التي نجمعها

نقوم بجمع الحد الأدنى من البيانات الضرورية لتشغيل البوت، مثل معرف السيرفر (Server ID) ومعرفات القنوات (Channel IDs) لتفعيل ميزات الأذكار التلقائية.

كيفية استخدام المعلومات

تُستخدم البيانات فقط لحفظ إعداداتك الخاصة مثل البريفكس المخصص وروم الأذكار المفضل، ولا يتم استخدامها لأي أغراض دعائية أو تجارية.

تخزين البيانات وحمايتها

يتم تخزين جميع البيانات محلياً داخل ملفات البوت configAzkar.json بشكل آمن تماماً، ولا نملك أي وصول لرسائل السيرفر أو بيانات الأعضاء الشخصية.

حقوق المستخدم

يمكن لمديري السيرفر حذف كافة البيانات المسجلة فوراً عبر طرد البوت أو استخدام أوامر الإيقاف المخصصة مثل +stop-azkar.
